# Graph Recommendation Engine

A graph-based social recommendation engine inspired by LinkedIn's "People You May Know" and Facebook friend suggestions.

## Features

- User Management
- Friend Connections
- Mutual Friend Detection
- Friends of Friends Candidate Generation
- Interest Similarity (Jaccard Similarity)
- Recommendation Ranking

## Tech Stack

### Backend

- Node.js
- TypeScript
- Express.js

### Databases (Planned)

- PostgreSQL
- Neo4j
- Redis

### Frontend (Planned)

- React
- Tailwind CSS
